rails3_haml_scaffold_generator by psynix

Very basic Rails 3 HAML scaffold generator

Essentially just a copy of the Rails 3 ERB generator with HAML replacements for the templates.

Dependencies

HAML

Install

1. rails 2. cd 3. Add "gem 'haml'" to Gemfile 4. Either 'gem install haml' or 'bundle install' 3. Edit config/application.rb and add the following: config.generators do |g| g.template_engine :haml end 4. git clone git://github.com/psynix/rails3_haml_scaffold_generator.git lib/generators 5. rails generate scaffold Stuff
